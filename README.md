# Mathematical Modeling Algorithms and Applications

This repository contains MATLAB code for the textbook **"Mathematical Modeling Algorithms and Applications"** (数学建模算法与应用, 3rd Edition) by Si Shoukui (司守奎) and Sun Xijing (孙玺菁).

## Directory Structure

```
chapter-XX/
├── sectionX.Y/
├── exerciseX/
└── ...
```
